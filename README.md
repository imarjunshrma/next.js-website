
# Next.js Website

A Beautiful Website with Next.js 13.4 and Bootstrap



## Features

- Variable fonts
- App Router
- Data Fetching
- Catche
- Styling
- Links
- Loading Page
- Error Page
- Images
- Css Modules
- use bootstrap with Next



## Run Locally

Clone the project

```bash
  git clone https://github.com/imarjunshrma/next.js_crud_api.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
